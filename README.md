Netsuko 是一款专为 Typecho 设计的博客主题，使用了Tailwind CSS构建。

> **注意** 本主题在开发过程中在编辑器中使用了 **Gemini 3.1 Pro** 作为AI辅助。

---
### 安装
1. Clone或下载Release至你的 Typecho 主题目录并解压，建议将文件夹重命名为 netsuko。
2. 登录 Typecho 后台，在“外观”设置中启用 Netsuko。

### 注意事项
1.本主题默认通过 CDN 加载 Tailwind CSS。运行时，请确保环境网络畅通。

2.进入“后台 -> 外观 -> 设置外观”，你可以自定义：
- **个人座右铭**：显示在头像下方的签名档。
- **社交链接**：配置 GitHub、Twitter、Telegram 等图标链接。
- **侧边栏链接**：支持 `名称|链接` 格式的快捷导航。
